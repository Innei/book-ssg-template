# 画板

## Excalidraw

支持 [Excalidraw](https://excalidraw.com/) 的内嵌。

例如：

```excalidraw
/draw/demo.json
```

语法为：

````
```excalidraw
/draw/demo.json
```
````

`/draw/demo.json` 是 excalidraw 的 json 虚拟化的数据的路径。你可以在 [Excalidraw](https://excalidraw.com/) 上进行绘制，然后导出为 json 格式。
